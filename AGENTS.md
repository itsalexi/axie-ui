# Axie UI Agent Guide

This repo contains a Next.js documentation app and a React component library.

## Structure

- `apps/docs`: documentation site, examples, recipes, templates, and public registry files.
- `packages/ui`: the `@axie/ui` component package.
- `packages/registry`: the component catalog used by docs, search, sidebar navigation, and registry generation.
- `registry/registry.json`: source shadcn registry metadata generated from the catalog.
- `apps/docs/public/r`: shadcn-compatible registry output generated by `pnpm registry:build`.

## Commands

- `pnpm dev`: build registry output, then run the docs app on port 8091.
- `pnpm registry:build`: regenerate `registry/registry.json` and static `/r/*.json` files.
- `pnpm typecheck`: run TypeScript checks for all workspaces.
- `pnpm build`: build registry output and the Next.js docs app.

## Component Rules

- Export every public component from `packages/ui/src/index.ts`.
- Add new component metadata to `packages/registry/src/component-catalog.json`.
- Add docs in `apps/docs/content/component-docs/<slug>.tsx`.
- Prefer existing Axie primitives, tokens, and motion classes before adding new styling.
- Keep component files source-copy friendly: avoid docs-only dependencies and keep helpers local or listed in registry metadata.
- If a component needs an external package, add it to `packages/ui/package.json` and to registry dependency metadata in `packages/registry/src/index.ts` and `scripts/write-registry.mjs`.

## Usage Pattern

Package install:

```tsx
import "@axie/ui/styles.css";
import { Button, Field, Input } from "@axie/ui";
```

Registry install after deploy:

```bash
pnpm dlx shadcn@latest registry add @axie=https://your-domain.com/r/{name}.json
pnpm dlx shadcn@latest add @axie/button
```

## Design Notes

- Axie UI is warm, tactile, compact, and product-focused.
- Use rounded controls, firm borders, muted surfaces, and restrained accent colors.
- Avoid decorative landing-page patterns in docs; make pages feel like real product surfaces.
- Preserve accessible native behavior for form controls and keyboard navigation.
